# Generate a single expert persona for a specific area

Generate a single expert persona for a specific area

## Usage

``` r
tempest_generate_single_persona(topic, area, existing_personas, config)
```

## Arguments

- topic:

  The research topic.

- area:

  The area of expertise needed.

- existing_personas:

  List of existing personas to avoid duplication.

- config:

  A `TempestConfig` object.

## Value

A persona list.
