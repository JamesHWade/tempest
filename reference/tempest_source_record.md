# Source record (S7) — typed replacement for the plain source list

Source record (S7) — typed replacement for the plain source list

## Usage

``` r
tempest_source_record(
  url,
  title = NA_character_,
  snippet = NA_character_,
  content_text = NA_character_,
  fetched_at = NA_character_,
  content_hash = NA_character_,
  meta = list(),
  id = NULL
)
```
