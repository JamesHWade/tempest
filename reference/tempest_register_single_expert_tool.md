# Register a single expert tool on a chat

Register a single expert tool on a chat

## Usage

``` r
tempest_register_single_expert_tool(chat, persona, session_manager, topic)
```

## Arguments

- chat:

  An ellmer chat object.

- persona:

  A persona object.

- session_manager:

  An `ExpertSessionManager` instance.

- topic:

  The research topic.

## Value

The chat object (invisibly).
