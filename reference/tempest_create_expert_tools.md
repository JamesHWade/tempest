# Create Expert Tools for All Personas

Creates ellmer tools for each expert persona and returns them as a list.

## Usage

``` r
tempest_create_expert_tools(personas, session_manager, topic)
```

## Arguments

- personas:

  List of persona objects.

- session_manager:

  An `ExpertSessionManager` instance.

- topic:

  The research topic.

## Value

A list of ellmer tools.
